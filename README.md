# Knok — Умная холодная рассылка

Минималистичное веб-приложение для B2B аутрича с AI-поиском лидов и автоматической рассылкой.

---

## ⚙️ Необходимые ключи (все бесплатные)

### Claude API Key (для AI-функций)

1. Зайдите на https://console.anthropic.com
2. Создайте аккаунт → API Keys → Create Key
3. Скопируйте ключ (начинается с `sk-ant-api03-`)
4. **Бесплатный кредит $5** при регистрации

### EmailJS (для отправки писем)

1. Зарегистрируйтесь на https://www.emailjs.com
2. **Add New Service** → выберите Gmail/Outlook/SMTP
3. Подключите свой почтовый ящик
4. **Email Templates** → Create New Template
   - В шаблоне используйте переменные: `{{to_email}}`, `{{to_name}}`, `{{subject}}`, `{{message}}`, `{{from_name}}`
5. Скопируйте:
   - Service ID (в разделе Email Services)
   - Template ID (в разделе Email Templates)
   - Public Key (Account → General → Public Key)
6. **Бесплатный план**: 200 писем/месяц

## 📋 Как работает

### Шаг 1 — Настройка профиля

- Опишите свою компанию и услуги
- Укажите целевую аудиторию (какие компании вам интересны)
- Введите Claude API Key

### Шаг 2 — Шаблон письма

- Напишите холодное письмо
- Используйте `{имя}`, `{компания}`, `{должность}` для персонализации
- AI поможет улучшить текст

### Шаг 3 — Email настройки

- Укажите EmailJS ключи
- Выберите дату начала рассылки
- Лимит: не более 30 писем в день (рекомендуется)

### Dashboard

- **Найти с AI** — Claude сгенерирует список из ~15 компаний вашей ЦА
- Отредактируйте и дополните список вручную
- Отправляйте письма одним кликом или запустите автокампанию

---

## 💡 Советы

- Чем подробнее описание ЦА — тем точнее AI подберёт компании
- Проверяйте email перед отправкой (AI угадывает паттерн, не всегда верно)
- Не превышайте 30-50 писем в день — риск попасть в спам
- Задержка 90+ секунд между письмами уменьшает шанс блокировки

---

## 🔒 Приватность

- Все данные хранятся **локально в браузере** (localStorage)
- API ключи не передаются третьим лицам
- Claude API запросы идут напрямую на api.anthropic.com

## ⚠️ Важно

Сайт работает как **статический HTML** — не нужен сервер или база данных.
Данные сохраняются в браузере. Для командного использования можно разместить
на GitHub Pages, Netlify или Vercel бесплатно.

**EmailJS шаблон** должен содержать как минимум эти переменные:

```
Кому: {{to_email}}
Имя: {{to_name}}
Тема: {{subject}}
Текст: {{message}}
```
